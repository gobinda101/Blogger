# 🌐 Blogger Tools & Codes Repository

यह रिपोजिटरी (Repository) विशेष रूप से **Blogger (Blogspot)** प्लेटफॉर्म के लिए बनाई गई है। इसमें आपको ब्लॉगर वेबसाइट को कस्टमाइज़ करने, स्पीड बढ़ाने और नए फीचर्स जोड़ने के लिए बेस्ट और रेडी-टू-यूज़ (Ready-to-Use) कोड्स मिलेंगे।

---

## 🚀 इस रिपोजिटरी में क्या-क्या है?

* 🎨 **Custom Widgets:** रीसेंट पोस्ट, पॉपुलर पोस्ट, और सुंदर सोशल मीडिया शेयर बटन्स।
* ⚡ **Optimization Codes:** ब्लॉगर की स्पीड बढ़ाने और SEO बेहतर करने के लिए स्क्रिप्ट्स।
* 💻 **CSS Styling:** ब्लॉगर टेम्पलेट्स को मॉडर्न और रिस्पॉन्सिव (Responsive) बनाने के लिए CSS कोड्स।
* 🛠️ **AdSense & SEO:** एडसेंस ऐड कोड्स को सही जगह सेटअप करने और मेटा टैग्स के कोड्स।

---

## 📂 फोल्डर स्ट्रक्चर (Folder Structure)

कोड को आसानी से ढूंढने के लिए इसे अलग-अलग फोल्डर्स में बांटा गया है:

* `/widgets` - ब्लॉगर के HTML/JavaScript गैजेट में यूज़ होने वाले कोड्स।
* `/css` - डिज़ाइन और स्टाइलिंग के लिए CSS कोड्स (Skin या Custom CSS के लिए)।
* `/seo-meta` - ब्लॉग के `<head>` सेक्शन में यूज़ होने वाले SEO टैग्स।
* `/scripts` - लोडिंग स्पीड और एक्स्ट्रा फीचर्स के लिए JavaScript।

## 🛠️ इन कोड्स को ब्लॉगर में कैसे इस्तेमाल करें?

ब्लॉगर में इन कोड्स को यूज़ करने के दो मुख्य तरीके हैं:

### तरीका 1: Layout (विजेट्स के लिए)
1. अपने **Blogger Dashboard** में जाएं।
2. **Layout** पर क्लिक करें।
3. जहां कोड दिखाना है, वहां **Add a Gadget** पर क्लिक करें और **HTML/JavaScript** चुनें।
4. इस रिपोजिटरी से कोड कॉपी करके वहां पेस्ट करें और **Save** कर दें।

### तरीका 2: Theme HTML (CSS, Meta Tags और Scripts के लिए)
1. ब्लॉगर डैशबोर्ड में **Theme** पर जाएं और **Edit HTML** पर क्लिक करें।
2. **CSS के लिए:** कोड को `]]> </b:skin>` या `</style>` टैग से ठीक पहले पेस्ट करें।
3. **Meta Tags/Scripts के लिए:** कोड को `<head>` टैग के नीचे या `</body>` टैग से ठीक पहले पेस्ट करें।
4. थीम को **Save** कर दें।

> ⚠️ **जरूरी सूचना:** थीम HTML में कोई भी बदलाव करने से पहले अपनी मौजूदा थीम का **Backup** जरूर ले लें।

## 🤝 योगदान कैसे करें (Contribution)

अगर आपके पास भी ब्लॉगर का कोई अच्छा कोड, विजेट या स्क्रिप्ट है, तो आप इस रिपोजिटरी में जोड़ सकते हैं:
1. इस प्रोजेक्ट को **Fork** करें।
2. अपनी नई स्क्रिप्ट या कोड को सही फोल्डर में ऐड करें।
3. एक **Pull Request (PR)** सबमिट करें।
